## 📞 Troubleshooting

### **Issue: Preview doesn't open**
```
Solution:
  1. Make sure you have the markdown file open
  2. Try: Right-click file tab → "Open Preview"
  3. Or install markdown preview extension
```

### **Issue: Print dialog doesn't appear**
```
Solution:
  1. Make sure you're in the PREVIEW pane (right side)
  2. Try clicking in the preview first
  3. Then press Ctrl+P
  4. Or right-click in preview → Print
```

### **Issue: "Save as PDF" option missing**
```
Solution:
  1. You might need a PDF printer installed
  2. Try: Settings → Printers → Add printer → "Microsoft Print to PDF"
  3. Or use online converter: cloudconvert.com/md-to-pdf
```

### **Issue: PDF formatting looks ugly**
```
Solution:
  1. Open print dialog again
  2. Set margins to "Minimal"
  3. Enable "Background graphics"
  4. Keep scale at 100%
  5. Re-save PDF
```
